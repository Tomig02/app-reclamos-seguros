
# CRUD REST API for insurance claims management 

## Description
This project is meant to be the backend for the frontend in [this other project](https://github.com/Tomig02/app-reclamos-seguros-frontend), it's meant to be simple and allow CRUD operations on the data for insurance claims at my job.  
Tech used: 
- C#
- ASP.NET core
- Newtonsoft.JSON 
- Sistem.Data.Sqlite
- xUnit

## Features
- [x] Data management for vehicle related claims
- [ ] Data management for real estate related claims
